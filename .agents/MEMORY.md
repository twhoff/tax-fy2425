# Tacit Knowledge

## User operating preferences

- For Australian tax work, the user does not want model-memory tax assertions. Verify tax-law claims from ATO or another credible source for the relevant financial year before relying on them.
- For this tax project, split broad work into small Beads epics and tasks to reduce drift and hallucination risk.
- Use deterministic scripts for calculations rather than mental arithmetic.
- Use `vscode_askQuestions` for quick household facts that would otherwise block progress.
- Keep local project knowledge in the `.agents` folder using the PARA memory layout when working on this repository.

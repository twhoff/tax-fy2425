# Document Renaming Epic

All documents must be renamed to follow the naming convention: `YYMMdd - <descriptive name>.ext`

## Convention
- Format: `YYMMdd - Description.ext`
- Example: `240819 - Amazon Prime Membership.pdf`

## Process
1. Use OCR/PDF text extraction to identify invoice/statement date
2. Rename file with date prefix
3. Ensure description is clear and concise
